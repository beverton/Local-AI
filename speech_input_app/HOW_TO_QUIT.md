# Wie beende ich die Speech Input App?

## Methode 1: Über das System-Tray-Menü (Empfohlen)

1. **Finde das Speech Input Icon** im System-Tray (rechts unten in der Taskleiste)
   - Falls nicht sichtbar: Klicke auf den Pfeil nach oben (^) um versteckte Icons anzuzeigen

2. **Rechtsklick** auf das Speech Input Icon

3. Klicke auf **"Beenden"** im Menü

Die App wird sauber beendet und alle Ressourcen werden freigegeben.

## Methode 2: Über den Task Manager

1. Öffne den **Task Manager** (Strg + Shift + Esc)
2. Suche nach **"python.exe"** oder **"speech_input.py"**
3. Rechtsklick → **"Task beenden"**

**Hinweis:** Diese Methode beendet die App abrupt - Methode 1 ist sauberer.

## Methode 3: Terminal/Batch-Fenster

Falls die App über ein Terminal gestartet wurde:
- Drücke **Strg + C** im Terminal-Fenster

## Tipp

Die App läuft im Hintergrund. Das Batch-Fenster schließt sich automatisch, wenn die App startet. Um die App zu beenden, nutze immer das System-Tray-Menü (Methode 1).
